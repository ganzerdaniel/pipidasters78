# pipidasters78

# 🚀 Сайт рекламы обучения программированию

> Проект представляет собой лендинг-сайт для продвижения обучения программированию.
> Основная цель — привлечение пользователей и демонстрация преимуществ курсов.

---

## 📌 Описание проекта

Этот сайт создан как учебный командный проект.
Он включает в себя современный дизайн, структуру страниц и базовую логику на JavaScript.

---

## 🧩 Структура проекта

```
📁 project
├── index.html      # Основная страница
├── styles.css      # Стили сайта
├── app.js          # Главная логика приложения
├── pages.js        # Управление страницами
├── data.js         # Данные и контент
└── README.md       # Документация проекта
```

---

## 👨‍💻 Команда проекта

В разработке участвовало **5 человек**:

* 🎨 **Верстка** — создание HTML структуры сайта
* 🎨 **Стилизация** — оформление и дизайн (CSS)
* ⚙️ **Логика приложения**:

  * `app.js` — основная логика работы сайта
  * `pages.js` — управление страницами и навигацией
  * `data.js` — хранение и обработка данных
